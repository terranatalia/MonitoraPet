### **Projeto Escolhido:  MonitoraPet**   
    
**Descrição:** Aplicativo com função de conectar Pet Shops e donos de animais domésticos e notificar os tutores do status do atendimento do animal sob cuidado dos estabelecimentos.

### **Funcionalidades Principais:**

1. Notificações de atualizações do status do atendimento, desde o check-in do animal até a retirada/entrega do animal em domicílio;
2. Acompanhamento da localização do animal em caso de entrega em domicílio;
3. Dashboard com Pet Shops locais registrados para fácil checagem de disponibilidade de atendimento;
4. Possibilidade de impulsionamento para publicidade (hub de pet shops locais);
5. Ficha de vacina, vermifugos e histórico de atendimento geral do pet (perfil pessoal);

**Público-alvo:** Pet Shops e donos de animais domésticos.

**Diferencial:** Múltiplas funcionalidades na mesma aplicação, praticidade para usuários.

## 2. Descrição da Personalização

### 2.1 Layout Principal

-   **container: { flex: 1 }**: Faz a View ocupar toda a tela.
    
-   **alignItems: 'center' e justifyContent: 'center'**: Garante que a aplicação ficará centralizada na tela.
    

### 2.2 Logo

-   **image: { width: 500, height: 300 }**: Padronização do tamanho da imagem nas telas de autenticação.
    

### 2.3 Inputs

-   **backgroundColor: '#fff'**: Fundo branco para destaque visual.
    
-   **borderWidth: 2, borderColor: 'Black'**: Borda sutil preta para melhor visualização.
    
-   **borderRadius: 18**: Cantos arredondados modernos.
    
-   **Width: '80%'**: Define que o input não ocupe a largura total da tela.
    

### 2.4 InputLabel

-   **alignItems: 'flex-start'**: Garante que o texto comece no início da linha.
    
-   **fontSize: 20**: Tamanho de fonte para facilitar a leitura.
    

### 2.5 Botão

-   **backgroundColor: '#2e6693'**: Cor definida para manter a identidade visual temática do projeto.
    
-   **alignSelf: 'center', alignItems: 'center'**: Mantém o botão e seu texto centralizados horizontalmente.
    
-   **borderRadius: 10**: Cantos arredondados.
